# Rice's Tabby List

A standalone tab list plugin for Purpur/Paper 1.21.4 - 1.21.11 (WILL SUPPORT OLDER VERSIONS IN FUTURE RELEASES)

## Build Instructions
```bash
./gradlew build

if you would like to change versions, please look at build.gradle
```

Output: `build/libs/RiceTabbyList.jar`

## Don't wanna build?
- Grab a .jar in releases

## Features
- Standalone tab list (no other plugins needed)
- TPS tracking
- OP count
- Domain display
- AFK integration (Essentials)
- Full color support (& and hex)
- PlaceholderAPI support

## Internal Placeholders
Use with `{}` in config.yml:
- {tps}, {op_count}, {players}, {max_players}, {domain}, {player}, {afk}, {afk_status}

## Commands
- /ricehead reload

See config.yml for configuration options.
